# Projeto

Para garantir que a 3G Store continuar sendo referência em qualidade e 
praticidade, estabelecer objetivos específicos para otimizar os processos e 
melhorar continuamente a experiência do cliente, integrar com APIs externas 
como o Mercado Livre e Shoppe. 

Assumir o compromisso de aprimorar a gestão do mercado, desde controlar o 
estoque de forma eficiente até agilizar o atendimento, assegurando um ambiente 
organizado e acessível a todos. 
Dessa forma, buscamos: 
    • Garantir uma gestão eficiente de produtos; 
    • Facilitar a reposição de mercadorias; 
    • Oferecer maior agilidade no controle;

Foi desenvolvido um software de controle de estoque inteligente, com recursos de 
gestão administrativa, capaz de se conectar a APIs externas, oferecendo uma 
solução completa e moderna para a gestão empresarial.
